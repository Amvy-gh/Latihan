Main Courses

Web Programming

Web Programming introduces students to the fundamental concepts and tools for designing and building websites. The course covers PHP, HTML, CSS, and JavaScript for creating responsive and interactive web pages. By the end, students will have practical skills in designing, implementing, and maintaining web systems, understanding the communication between client browsers and web servers.

Object Oriented Programming

Object-Oriented Programming (OOP) introduces the principles of software design using objects and classes. Students will learn key OOP concepts such as inheritance, encapsulation, and polymorphism. Through hands-on projects, learners will gain practical experience in solving real-world problems using OOP techniques, preparing them for advanced programming challenges.

Data Structure and Algorithm

Data Structures introduces students to the fundamental concepts of organizing and managing data efficiently. The course covers essential structures such as arrays, linked lists, stacks, queues, trees, and graphs. Students will learn how to select and apply appropriate data structures to solve complex computational problems, optimizing performance and resource usage.
